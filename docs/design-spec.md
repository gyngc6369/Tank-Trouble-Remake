# 设计规范文档 — 《坦克动荡》

> 版本：v1.0 | 日期：2026-06-26

## 1. 视觉设计

### 1.1 设计原则

- **简洁几何风格**：所有图形均为基本几何形状
- **白底黑墙**：高对比度，清晰可辨
- **色彩区分**：不同玩家用不同颜色标识
- **最小化干扰**：UI 简洁，信息精炼

### 1.2 色彩方案

| 元素 | 颜色 | RGB |
|------|------|-----|
| 背景 | 白色 | `(255, 255, 255)` |
| 墙壁线条 | 黑色 | `(0, 0, 0)` |
| 玩家1 车身 | 蓝色 | `(60, 100, 220)` |
| 玩家1 炮管 | 深蓝 | `(30, 60, 160)` |
| 玩家2 车身 | 红色 | `(220, 60, 60)` |
| 玩家2 炮管 | 深红 | `(160, 30, 30)` |
| AI 车身 | 橙色 | `(220, 150, 50)` |
| AI 炮管 | 深橙 | `(160, 100, 20)` |
| 玩家1 炮弹 | 蓝色 | `(60, 100, 220)` |
| 玩家2 炮弹 | 红色 | `(220, 60, 60)` |
| AI 炮弹 | 橙色 | `(220, 150, 50)` |
| UI 文字 | 深灰 | `(40, 40, 40)` |
| 遮罩层 | 半透明黑 | `(0, 0, 0, 128)` |

## 2. 布局设计

### 2.1 屏幕分区

```
┌────────────────────── 960 px ──────────────────────┐
│ 顶部 UI 栏 (40px)                                   │ 744 px
│ P1分数 │ P1弹药 │ 回合信息 │ P2/AI弹药 │ P2/AI分数  │
├────────────────────────────────────────────────────┤
│                                                    │
│              地图区域 (704px)                       │
│            15×11格 × 64px/格                       │
│                                                    │
└────────────────────────────────────────────────────┘
```

### 2.2 地图网格坐标

- 左上角格子 (0,0) 的左上角像素位于 `(0, UI_BAR_HEIGHT)` = `(0, 40)`
- 右下角格子 (14,10) 的右下角像素位于 `(960, 744)`
- 每个格子 64×64 px
- 墙线在格子的边界上，4px 粗

### 2.3 坦克绘制

```
         ┌─────┐
         │炮管 │ ← 4×16px，从车身前边中点伸出
    ┌────┼─────┤
    │    │     │
    │ 车身     │ ← 28×22px，矩形
    │    │     │
    └────┼─────┘
         │ ← 炮管伸出部分不计入碰撞体积
```

- 车身：28×22 px 矩形，中心为坦克坐标
- 炮管：4×16 px 矩形，从车身前边（相对于旋转方向）中点出发
- 整体以车身中心为轴旋转 `angle` 弧度
- 渲染顺序：先车身，后炮管（炮管在车身上层）

### 2.4 炮弹绘制

- 圆形，半径 6px
- 颜色与发射者相同
- 无拖尾效果（简洁优先）

## 4. UI 元素设计

### 4.1 顶部状态栏

```
┌──────────────────────────────────────────────────────┐
│ P1 分数:3  ●●●●○○○  第3回合  ○○○●●●●  P2 分数:2    │
└──────────────────────────────────────────────────────┘
```

- 弹药显示：7 个小圆点（实心=有弹，空心=空）
- 回合信息：居中显示当前回合数
- 字体：使用 Pygame 默认字体，字号 20-24px

### 4.2 回合结束覆盖层

```
    ┌──────────────────────────┐
    │                          │
    │     🏆 玩家1 赢得此回合！  │
    │                          │
    │     玩家1  3  :  2  玩家2 │
    │                          │
    │     下一回合 3...         │
    │                          │
    └──────────────────────────┘
```

- 半透明黑色背景遮罩
- 白字或亮色文字
- 3 秒倒计时

### 4.3 游戏结束界面

```
    ┌──────────────────────────┐
    │                          │
    │     🏆 玩家1 获得最终胜利！│
    │                          │
    │     最终比分  5  :  3    │
    │                          │
    │   [R] 再来一局           │
    │   [M] 返回主菜单         │
    │                          │
    └──────────────────────────┘
```

## 6. 交互反馈

| 用户操作 | 反馈 |
|----------|------|
| 菜单项悬停 | 按钮高亮（浅灰填充） |
| 菜单项点击 | 播放短促点击音？/ 直接切换 |
| 坦克射击 | 射击音效 + 炮口闪光？+ 炮弹生成 |
| 坦克被击中 | 击毁音效 + 坦克闪烁消失 |
| 子弹弹墙 | 无特殊反馈（自然反弹） |
| 子弹消失 | 静默消失 |
| 回合结束 | 3 秒倒计时 + 分数更新 |
